* State of Err 2026-06-22

Overall err has a few basic goals:

1. Provide high quality user respecting metrics tracking for hiking, cycling and any human powered travel.
2. Making the above approachable and rewarding for people who are not on the high performance athletics path.
3. Providing this via a community supported, free and open software package.


Err is already pretty useful to me today but to make all of these happen there needs to be some more. A lot of this application has been generated by me investing a month into claude code. Its really shitty hard to parse code for a human and probably has a world of bugs. The application also is hard to read data from. The csv and GPX files work for me, and are portable but are a mess.
